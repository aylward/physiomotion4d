Datasets
========

DirLab 4D-CT
------------

Public benchmark for respiratory motion:

* 10 cases with exhale and inhale phases
* 300 landmark points for validation
* `Website <https://www.dir-lab.com/>`_

Slicer-Heart-CT
---------------

Cardiac CT examples from 3D Slicer:

* Multiple cardiac phases
* Various pathologies
* `3D Slicer Website <https://www.slicer.org/>`_
