====================
Visualization Guide
====================

Guide to visualizing results in Omniverse and other tools.

NVIDIA Omniverse
================

1. Install NVIDIA Omniverse
2. Launch USD Composer or USD Presenter
3. Open generated USD file
4. Press Play to view animation

Other Tools
===========

* **usdview**: Command-line USD viewer
* **PyVista**: Python-based 3D visualization
* **ParaView**: Scientific visualization

See :doc:`../examples` for visualization examples.
