=======================
USD Conversion Guide
=======================

Guide to creating USD files for NVIDIA Omniverse.

USD Format
==========

Universal Scene Description (USD) is Pixar's format for 3D interchange,
used by NVIDIA Omniverse for visualization.

Conversion Tools
================

* **ConvertVTK4DToUSD**: Convert VTK time series to USD
* **USDTools**: Merge and manipulate USD files
* **USDAnatomyTools**: Apply anatomical materials

See :doc:`../api/utilities` for detailed documentation.
