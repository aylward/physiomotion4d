====================
Segmentation Guide
====================

Guide to using AI-based segmentation in PhysioMotion4D.

Available Methods
=================

* **TotalSegmentator**: Fast, 104 anatomical structures
* **VISTA-3D**: High accuracy, foundation model
* **VISTA-3D NIM**: Cloud-based inference
* **Ensemble**: Combines multiple methods

See :doc:`../api/segmentation` for API reference.
